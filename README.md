# LL1 Parser
By Mark Ureta
@ 2025

---

This is a simple LL(1) Parser implementation that recognises the MiniLisp grammar.

# Execution
## MacOS
```bash
./run.sh
```

or for running tests
```bash
./run.sh -t
./run.sh --test
```

## Windows
```CMD
run.bat
```

or for running tests

```CMD
run.bat -t
run.bat --test
```
